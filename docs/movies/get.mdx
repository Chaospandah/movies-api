# Get All Movies

## Endpoint
`/movies`

### Response (200 OK)

```json
  {
    "id": 1,
    "title": "Interstellar",
    "release_year": 2014,
    "genres": ["Sci-Fi", "Drama"],
    "rating": 9.0,
    "summary": "Earth is dying and a former pilot has to travel through a wormhole to save it."
  }
```

## Endpoint

`/movies/{id}`

### Response (200 OK)

```json
{
  "id": 1,
  "title": "Interstellar",
  "release_year": 2014,
  "genres": ["Sci-Fi", "Drama"],
  "rating": 9.0,
  "summary": "Earth is dying and a former pilot has to travel through a wormhole to save it."
}
```

## Error Responses
- 404 Not Found: Movie with the specified ID does not exist.
```json
{
  "detail": "Movie not found"
}
```


## Filtering Movies by Genre
You can filter movies by genre using the `genre` query parameter.
### Example Request
`/movies?genre=Action`
### Response (200 OK)
```json
  {
    "id": 2,
    "title": "Mad Max: Fury Road",
    "release_year": 2015,
    "genres": ["Action", "Adventure"],
    "rating": 8.1,
    "summary": "In a post-apocalyptic wasteland, Max teams up with Furiosa to escape a tyrannical warlord."
  }
```
