The Movies resource represents a film in the database. You can create, retrieve, update, 
and delete movies using the endpoints in this section.

## Available Endpoints

- [List all movies](./get.mdx)
- [Get a movie by ID](./get.mdx)
- [Create a movie](./create.mdx)
- [Update a movie](./update.mdx)
- [Delete a movie](./delete.mdx)
